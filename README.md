# qaoa-maxcut-cyclic-graphs
QAOA for MaxCut on cyclic graphs | Testing the hypothesis that convergence requires n/2 layers | Comparison of 4 optimizers (Adagrad, Adam, GD, COBYLA)

Алгоритм QAOA (Quantum Approximate Optimization Algorithm) использует гибридную квантово-классическую схему для приближенного решения комбинаторной задачи MaxCut на циклическом графе.

Квантовая схема состоит из трех основных этапов:
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  Инициализация  │ ──► │  p слоев QAOA   │ ──► │   Измерение     │
│   |+⟩^{⊗n}      │     │  (γ, β)         │     │   (sample)      │
└─────────────────┘     └─────────────────┘     └─────────────────┘

### 1. Инициализация квантового состояния

`for i in range(n):`
    `qml.Hadamard(wires=i)`
    
Первый шаг алгоритма — подготовка равномерной суперпозиции всех возможных битовых строк. Для этого к каждому кубиту, исходно находящемуся в состоянии ∣0⟩, применяется **гейт Адамара H**:

```math
H|0\rangle = \frac{|0\rangle + |1\rangle}{\sqrt{2}} = |+\rangle
```
В результате для системы из \( n \) получается состояние \[|\psi_0\rangle\], представляющее собой равномерную суперпозицию всех \( 2^n \) битовых строк: 

```math
|\psi_0\rangle = H^{\otimes n} |0\rangle^{\otimes n} = \frac{1}{\sqrt{2^n}} \sum_{x \in \{0,1\}^n} |x\rangle
```

то есть все возможные решения задачи MaxCut присутствуют в квантовом состоянии с одинаковой амплитудой ```math \1/ sqrt(( 2^n \)) ```

### 2. Квантовая интерференция

Алгоритм стартует с равным весом всех возможных решений.  
Затем слои QAOA (чередование операторов \( U_C(\gamma) \) и \( U_B(\beta) \)) создают **интерференцию**:
- Битовые строки, дающие хорошие разрезы, усиливаются
- Плохие решения подавляются

### 3. Измерение

При измерении квантовое состояние коллапсирует в битовую строку.  
Благодаря интерференции, вероятность получить хорошее решение значительно выше, чем при случайном угадывании.
