# qaoa-maxcut-cyclic-graphs
QAOA for MaxCut on cyclic graphs | Testing the hypothesis that convergence requires n/2 layers | Comparison of 4 optimizers (Adagrad, Adam, GD, COBYLA)

## Описание 

В данном проекте исследуется применение квантового алгоритма QAOA (Quantum Approximate Optimization Algorithm) для решения задачи MaxCut на циклических графах.

**Основная цель работы:** эмпирически проверить гипотезу о том, что для сходимости QAOA на циклическом графе с n вершинами требуется p = n/2 слоев. Дополнительно проводится сравнительный анализ эффективности четырех классических оптимизаторов — Adagrad, Adam, классического градиентного спуска (GD) и безградиентного метода COBYLA — при подборе параметров γ и β в гибридном квантово-классическом цикле.

Эксперименты проводятся для циклических графов с числом вершин n = 4, 6, 8, 10 и 12. Для каждого графа QAOA запускается с последовательным увеличением числа слоев p до достижения сходимости. Критерием сходимости служит достижение целевой энергии E_target = n для четных n или E_target = n-1 для нечетных n с точностью ε = 0.1. Для каждого значения p параметры γ и β оптимизируются всеми четырьмя методами, после чего сравниваются скорость сходимости и качество полученного решения.

Ожидаемые результаты включают подтверждение или опровержение гипотезы о линейной зависимости p = n/2, а также выявление наиболее эффективного оптимизатора для данной задачи по таким критериям, как скорость сходимости, финальная энергия и стабильность результатов.

### 1. Инициализация квантового состояния

`for i in range(n):`
    `qml.Hadamard(wires=i)`
    
Первый шаг алгоритма — подготовка равномерной суперпозиции всех возможных битовых строк. Для этого к каждому кубиту, исходно находящемуся в состоянии ∣0⟩, применяется **гейт Адамара H**:

```math
H|0\rangle = \frac{|0\rangle + |1\rangle}{\sqrt{2}} = |+\rangle
```
В результате для системы из \( n \) получается состояние \[|\psi_0\rangle\], представляющее собой равномерную суперпозицию всех \( 2^n \) битовых строк: 

```math
|\psi_0\rangle = H^{\otimes n} |0\rangle^{\otimes n} = \frac{1}{\sqrt{2^n}} \sum_{x \in \{0,1\}^n} |x\rangle
```

то есть все возможные решения задачи MaxCut присутствуют в квантовом состоянии с одинаковой амплитудой `1/√(2ⁿ)`.
